基础语法 逐条对照

29 个最基础的语法概念，每个一张独立卡片，左 Java 右 Python，附关键差异说明。专为打开就能查、查完就能用而设计。

01变量声明

Java 必须显式声明类型；Python 直接赋值，类型由值决定。这是最根本的差异之一。

基本声明并赋值 VARIABLE
Java
int    x = 10;
double y = 3.14;
String s = "hi";
boolean b = true;
Python
x = 10        # 自动是 int
y = 3.14      # 自动是 float
s = "hi"      # 自动是 str
b = True      # 自动是 bool
无类型声明，无 ;。变量的"类型"是它当前指向对象的类型，不是变量本身的属性。
类型注解（可选） VARIABLE
Java 类型必须
int x = 10;
String s = "hi";
List<Integer> xs = new ArrayList<>();
Python 类型可选
x: int = 10
s: str = "hi"
xs: list[int] = []
# 注解只是给 IDE 和 mypy 看的
# 运行时不强制
Python 的类型注解 不影响运行，只用于静态检查。x: int = "abc" 不会报错。
声明但未初始化 VARIABLE
Java
int x;          // 默认 0
String s;       // 默认 null
// 局部变量必须显式赋值才能用
Python
x = None        # 必须给值
# 单纯写 x 会 NameError
# 一般用 None 表示"未初始化"
Python 没有"声明但不赋值"的概念。变量第一次赋值就是创建。
多重赋值 / 交换 VARIABLE
Java
int a = 1, b = 2, c = 3;

// 交换需要临时变量
int tmp = a;
a = b;
b = tmp;
Python
a, b, c = 1, 2, 3

# 交换：一行搞定
a, b = b, a

# 多变量来自函数
x, y = get_pair()
常量 VARIABLE
Java
final int MAX = 100;     // 不可改
static final int CAPACITY = 10;
Python
MAX = 100        # 约定：大写 = 常量
# 但语法上仍可改！

# 想要"真常量"用 typing.Final
from typing import Final
CAPACITY: Final = 10
Python 没有真正的不可变变量，全靠命名约定（大写）。Final 只是给 mypy 看的。
作用域 VARIABLE
Java
if (cond) {
    int x = 1;     // x 仅在 {} 内可见
}
// x 这里访问不到
Python
if cond:
    x = 1          # x 在函数级别可见
print(x)           # 这里照样能用！
Python 没有块作用域。变量作用域是 函数级别（或模块级别）。if/for/while 不会创建新作用域。这是 Java 程序员常踩的坑。

02基本数据类型

Python 没有"基本类型"（primitive），所有值都是对象。int 也是对象，可以调方法。

整数 TYPE
Java
byte  b = 127;          // 8 位
short s = 32000;        // 16 位
int   i = 2000000000;   // 32 位
long  l = 9000000000L;  // 64 位
// 溢出会环绕，要大数用 BigInteger
Python
x = 100
big = 10 ** 100     # 永远不溢出
huge = 2 ** 1000    # 任意精度，免费

# 数字下划线分隔（可读）
million = 1_000_000

# 不同进制
hex_n = 0xff
bin_n = 0b1010
oct_n = 0o17
Python 的 int 是 任意精度，永远不溢出。Java 需要 BigInteger。
浮点数 TYPE
Java
float  f = 3.14F;     // 32 位
double d = 3.14;      // 64 位
// 默认字面量是 double
Python
x = 3.14            # float（统一是 64 位）
y = 1.5e3           # 1500.0
z = float('inf')    # 无穷大
n = float('nan')    # 非数

# 没有 float 类型
# 浮点精度问题一样存在
0.1 + 0.2  # 0.30000000000000004
只有一种浮点类型（相当于 Java 的 double）。精确十进制用 Decimal。
布尔 TYPE
Java
boolean b = true;
boolean done = false;
// 必须 boolean 才能用在 if
Python
b = True            # 注意首字母大写
done = False

# Python 中 bool 是 int 的子类！
True + True         # 2
True == 1           # True

# 任何对象都能用在 if（真值判断）
Python 里 True / False 首字母大写，且 bool 继承自 int，所以 True == 1。
字符 TYPE
Java
char c = 'A';        // 单引号
char d = '中';
int code = (int) c;  // 65
Python
c = 'A'             # 单双引号都行
d = '中'
# 没有 char 类型！
# 'A' 就是长度为 1 的 str
type('A')           # <class 'str'>
ord('A')            # 65
chr(65)             # 'A'
Python 没有 char 类型，单字符就是长度 1 的字符串。

数字与字符串互转 CONVERT
Java
int n = Integer.parseInt("123");
double d = Double.parseDouble("1.5");
String s = Integer.toString(123);
String x = String.valueOf(1.5);
Python
n = int("123")       # 123
d = float("1.5")     # 1.5
s = str(123)         # '123'
x = str(1.5)         # '1.5'

# int 还可以转换其他进制
int("ff", 16)        # 255
int("1010", 2)       # 10
Python 用类型名当函数：int()、str()、float()、bool()、list()。
数字间转换 CONVERT
Java
int    i = (int) 3.7;       // 3 截断
double d = (double) 5;       // 5.0
long   l = (long) i;
Python
i = int(3.7)        # 3 截断
d = float(5)        # 5.0
# 没有 long，int 自动扩展

04null / None 对照

空值表示 NULL
Java
String s = null;
if (s == null) ...
if (s != null && !s.isEmpty()) ...

// 现代代码用 Optional
Optional<String> o = Optional.empty();
Python
s = None            # ← 注意大写！
if s is None: ...    # ← 用 is，不用 ==
if s is not None: ...
if s: ...            # 同时检查非 None 和非空字符串
                     # （但 0/空集合也是假，要小心）
关键：和 None 比较 一定用 is，不用 ==。None 是单例，is 比 == 快且更明确。
可空类型注解 NULL
Java
@Nullable String name;
Optional<User> findUser(int id);
Python
from typing import Optional

name: Optional[str] = None    # 等价于 str | None
def find_user(uid: int) -> Optional[User]: ...

# 3.10+ 推荐用管道
name: str | None = None
def find_user(uid: int) -> User | None: ...

05算术运算

大部分运算符相同，但 /、**、++ 有重要差异。

基本四则 ARITH
Java
int sum = 5 + 3;     // 8
int sub = 5 - 3;     // 2
int mul = 5 * 3;     // 15
int div = 5 / 2;     // 2 ← 整数相除是整除
double d = 5.0 / 2;  // 2.5
int mod = 5 % 2;     // 1
Python
sum = 5 + 3      # 8
sub = 5 - 3      # 2
mul = 5 * 3      # 15
div = 5 / 2      # 2.5  ← / 永远是真除法！
fdiv = 5 // 2    # 2    ← // 才是整除
mod = 5 % 2      # 1
关键：Python 的 / 永远返回 float，// 才是整除。这是最常见的差异。
幂、绝对值、最值 ARITH
Java
double p = Math.pow(2, 10);    // 1024.0
int    a = Math.abs(-5);
int    m = Math.max(3, 5);
int    n = Math.min(3, 5);
double r = Math.sqrt(2);
Python
p = 2 ** 10       # 1024 ← 内建运算符
a = abs(-5)       # 5  ← 内建函数
m = max(3, 5)
n = min(3, 5)

import math
r = math.sqrt(2)
自增自减 ARITH
Java
i++;    // 后置自增
++i;    // 前置自增
i--;
i += 1;   // 也行
Python
# Python 没有 ++ / --！
i += 1     # 必须这样
i -= 1
i *= 2
i //= 3
i **= 2
Python 没有 ++ 运算符。只能用 i += 1。新手最常踩这个坑。

06比较与逻辑运算

比较运算 COMPARE
Java
a == b;   // 基本类型比值；对象比引用！
a != b;
a < b;  a > b;  a <= b;  a >= b;

// 对象比内容
str1.equals(str2);
list1.equals(list2);
Python
a == b      # 比内容（默认）
a != b
a < b; a > b; a <= b; a >= b

# 比身份（是否同一个对象）
a is b
a is not b

# 链式比较（Java 没有！）
if 0 < x < 100: ...
if a == b == c: ...
关键：== 在 Java 比对象引用，在 Python 比内容（调 __eq__）。Python 想比身份用 is。
另外 Python 的 链式比较 是杀手特性：0 < x < 100 直接可用。
逻辑运算 LOGIC
Java
a && b
a || b
!a

// 必须 boolean 类型
Python
a and b
a or b
not a

# 任何对象都能用，按真值判断
# 短路求值，返回值不是 bool

x = name or "default"   # 常用模式
x = a and a.field        # 安全访问
Python 的 and / or 返回的是 原对象，不是 bool。"" or "fallback" 返回 "fallback"。
成员判断 LOGIC
Java
list.contains(x)
str.contains("a")
map.containsKey(k)
set.contains(x)
Python
x in list_      # 通用！
"a" in str_
k in dict_      # 检查键
x in set_

# 否定
x not in list_
in 是运算符，不是方法，对所有可迭代/容器都适用。

07赋值与三元表达式

三元表达式 CONDITION
Java
String s = (age >= 18) ? "adult" : "minor";

int max = (a > b) ? a : b;
Python
s = "adult" if age >= 18 else "minor"

max_v = a if a > b else b
关键：Python 的三元 语序不同：值1 if 条件 else 值2。读起来更像英语。
复合赋值 ASSIGN
Java
a += 1;
a -= 1;
a *= 2;
a /= 2;
a %= 3;
Python
a += 1
a -= 1
a *= 2
a /= 2
a //= 2     # 整除赋值
a %= 3
a **= 2     # 幂赋值
海象运算符 := （3.8+） ASSIGN
Java
// 类似但更受限
String line;
while ((line = reader.readLine()) != null) {
    process(line);
}
Python
# 在表达式里赋值
while (chunk := f.read(1024)):
    process(chunk)

# 推导式里更有用
[y for x in data if (y := f(x)) > 0]

08字符串操作

字符串字面量 STRING
Java
String a = "hello";
char   c = 'a';   // ← 不同类型

String multi = """
    line1
    line2
    """;        // 14+

String esc = "C:\\Users";  // 需转义
Python
a = "hello"
b = 'hello'        # 单双引号等价
c = 'a'            # 还是 str（无 char 类型）

multi = """
    line1
    line2
"""

raw = r"C:\Users"  # 原始字符串，不转义
fstr = f"Hi {a}"   # f-string
字符串拼接 STRING
Java
String s = "Hello, " + name + "! Age: " + age;
String t = String.format("Hi %s, age %d", name, age);
StringBuilder sb = new StringBuilder()
    .append("a").append("b");
Python
s = "Hello, " + name + "! Age: " + str(age)   # 必须先转 str
s = f"Hello, {name}! Age: {age}"   # ← 推荐！
s = "Hi {}, age {}".format(name, age)         # 老式

# 大量拼接用 join
result = "".join(parts)
推荐：永远用 f-string。Python 字符串拼接 + 时数字必须先转 str()，f-string 自动处理。
f-string 格式说明符 STRING
Java
String.format("%.2f", 3.14159);    // "3.14"
String.format("%10s", "ab");       // "        ab"
String.format("%-10s|", "ab");     // "ab        |"
String.format("%,d", 1234567);     // "1,234,567"
Python
f"{3.14159:.2f}"     # '3.14'
f"{'ab':>10}"        # '        ab'
f"{'ab':<10}|"       # 'ab        |'
f"{'ab':^10}|"       # '    ab    |'  居中
f"{1234567:,}"       # '1,234,567'
f"{255:#x}"          # '0xff'
f"{255:08b}"         # '11111111'

# 调试神器
x = 42
f"{x=}"              # 'x=42'
长度、大小写、查找、替换 STRING
Java
s.length();
s.toUpperCase();
s.toLowerCase();
s.contains("x");
s.indexOf("x");
s.replace("a", "b");
s.startsWith("a");
s.endsWith("z");
s.trim();
s.split(",");
String.join(",", arr);
Python
len(s)               # 全局函数！
s.upper()
s.lower()
"x" in s             # in 运算符
s.find("x")          # 找不到返回 -1
s.replace("a", "b")
s.startswith("a")
s.endswith("z")
s.strip()
s.split(",")
",".join(arr)        # ← 连接符调方法！
注意：len() 是 全局函数，不是方法。join 是 连接符（字符串） 调方法，不是列表。
字符串切片（Java 没有） STRING
Java
s.substring(1, 4);          // 1..4 (不含 4)
s.substring(0, s.length()-1);
String reversed = new StringBuilder(s).reverse().toString();
s.charAt(0);
s.charAt(s.length() - 1);
Python
s[1:4]            # 1..4 (不含 4)
s[:-1]            # 去掉最后一个字符
s[::-1]           # 反转！
s[0]              # 第一个字符
s[-1]             # 最后一个字符
s[::2]            # 每隔一个取
s[-3:]            # 最后 3 个
杀器：切片语法 s[start:stop:step] 比 substring 强大得多，且支持负索引。
字符串重复 STRING
Java
"abc".repeat(3);          // "abcabcabc"
Python
"abc" * 3           # 'abcabcabc'  ← 运算符！
"-" * 50            # 画分割线

09条件语句

if / else CONTROL
Java
if (x > 0) {
    System.out.println("positive");
} else {
    System.out.println("non-positive");
}
Python
if x > 0:
    print("positive")
else:
    print("non-positive")
# 1. 条件不要括号
# 2. 用 :  + 缩进，不用 {}
if / else if / else CONTROL
Java
if (score >= 90) {
    grade = "A";
} else if (score >= 80) {
    grade = "B";
} else if (score >= 60) {
    grade = "C";
} else {
    grade = "F";
}
Python
if score >= 90:
    grade = "A"
elif score >= 80:        # ← elif，不是 else if
    grade = "B"
elif score >= 60:
    grade = "C"
else:
    grade = "F"
用 elif，不是 else if。

10循环

基本 for（C 风格） LOOP
Java
for (int i = 0; i < 10; i++) {
    System.out.println(i);
}

for (int i = 5; i < 20; i += 2) {
    System.out.println(i);
}
Python
for i in range(10):       # 0..9
    print(i)

for i in range(5, 20, 2): # 5,7,9,...,19
    print(i)

# 倒序
for i in range(9, -1, -1): print(i)
Python 没有 C 风格 for。用 range(start, stop, step)。
遍历集合 LOOP
Java
for (String s : list) {
    System.out.println(s);
}

for (Map.Entry<String, Integer> e : map.entrySet()) {
    System.out.println(e.getKey() + "=" + e.getValue());
}
Python
for s in my_list:
    print(s)

for k, v in my_dict.items():    # 解包
    print(f"{k}={v}")

for k in my_dict:        # 默认遍历键
    print(k)
for v in my_dict.values():
    print(v)
同时取索引和元素 LOOP
Java
for (int i = 0; i < list.size(); i++) {
    String s = list.get(i);
    System.out.println(i + ": " + s);
}
Python
for i, s in enumerate(my_list):
    print(f"{i}: {s}")

# 从 1 开始计数
for i, s in enumerate(my_list, start=1):
    print(f"{i}: {s}")
enumerate 是 Python 必学函数，比手动维护索引优雅 10 倍。
同时遍历多个序列 LOOP
Java
for (int i = 0; i < names.size(); i++) {
    String name = names.get(i);
    int    age  = ages.get(i);
    System.out.println(name + ": " + age);
}
Python
for name, age in zip(names, ages):
    print(f"{name}: {age}")

# 任意多个
for a, b, c in zip(list1, list2, list3): ...
while LOOP
Java
while (running) {
    doStuff();
}

do {
    doStuff();
} while (cond);
Python
while running:
    do_stuff()

# Python 没有 do-while！
# 模拟：
while True:
    do_stuff()
    if not cond:
        break
break / continue / 循环 else LOOP
Java
for (int i : nums) {
    if (i < 0) continue;
    if (i > 100) break;
    process(i);
}
// Java 没有循环 else
Python
for i in nums:
    if i < 0: continue
    if i > 100: break
    process(i)
else:
    # 仅在循环正常结束（没 break）才执行
    print("all processed")
独有：Python 的 for/while else，只有循环正常结束（没 break）才执行。常用于"找到了就 break，没找到就 else"。

11switch / match

基本分支 SWITCH
Java
switch (day) {
    case 1: name = "Mon"; break;
    case 2: name = "Tue"; break;
    case 3: name = "Wed"; break;
    default: name = "?";
}

// Java 14+ 表达式形式
String name = switch (day) {
    case 1 -> "Mon";
    case 2 -> "Tue";
    default -> "?";
};
Python 3.10+
match day:
    case 1: name = "Mon"
    case 2: name = "Tue"
    case 3: name = "Wed"
    case _: name = "?"     # _ = 默认

# Python 的 match 比 Java switch 强大：
match point:
    case (0, 0): print("origin")
    case (x, 0): print(f"x-axis: {x}")
    case (x, y) if x == y: print("diagonal")
    case [_, *rest]: print(f"list, rest={rest}")
Python match 是"结构化模式匹配"，可以解构对象/列表/字典，比 switch 强大得多。

12函数定义

Python 用 def 定义函数，无类型声明，可加可选注解。

最简函数 FUNCTION
Java（必须在类里）
public class Util {
    public static int add(int a, int b) {
        return a + b;
    }
}

int n = Util.add(1, 2);
Python（顶层即可）
def add(a, b):
    return a + b

n = add(1, 2)

# 可选类型注解
def add(a: int, b: int) -> int:
    return a + b
Python 函数不需要包在类里。文件本身就是模块。
无返回值 FUNCTION
Java
public static void log(String msg) {
    System.out.println(msg);
}
Python
def log(msg):
    print(msg)
    # 无 return 等价于 return None

# 类型注解写 None
def log(msg: str) -> None:
    print(msg)
文档字符串（替代 Javadoc） FUNCTION
Java
/**
 * 加法
 * @param a 第一个数
 * @param b 第二个数
 * @return 和
 */
public int add(int a, int b) { ... }
Python
def add(a, b):
    """
    加法

    Args:
        a: 第一个数
        b: 第二个数
    Returns:
        和
    """
    return a + b

help(add)    # 显示 docstring

13函数参数（5 种形式）

默认参数 PARAM
Java 需重载
public String greet(String name) {
    return greet(name, "Mr");
}
public String greet(String name, String prefix) {
    return prefix + ". " + name;
}
Python
def greet(name, prefix="Mr"):
    return f"{prefix}. {name}"

greet("Tom")              # 'Mr. Tom'
greet("Tom", "Dr")        # 'Dr. Tom'
greet("Tom", prefix="Dr") # 关键字传递
Python 默认参数 = 不需要重载。但默认值必须是不可变对象（陷阱：def f(x=[]) 是 bug）。
关键字参数 PARAM
Java 不支持
// 只能按位置
connect("localhost", 8080, true, 30);
// 可读性差！必须看签名才知道每个参数啥意思
Python
def connect(host, port, ssl=False, timeout=30):
    ...

# 按关键字传，可读性极佳
connect(host="localhost", port=8080, ssl=True, timeout=30)
connect("localhost", 8080, ssl=True)   # 混合也行
可变参数 *args PARAM
Java
public int sum(int... nums) {
    int total = 0;
    for (int n : nums) total += n;
    return total;
}
sum(1, 2, 3, 4);
Python

最重要的 5 句话
  1. 变量没有类型，对象有类型——所以不需要声明类型。
  2. 函数/类/模块都是对象——可以赋值、传参、返回。
  3. 变量是"标签"不是"盒子"——a = b 不复制，改 b 会影响 a（如果可变）。
  4. self 是显式参数，不是隐式 this——所有方法第一个参数都是它。
  5. None 用 is 比，/ 是真除，i++ 不存在——三个最常见的"想当然错误"。
