## 2. AI 栈信创替代

| 层 | 国际栈 | 信创替代 |
|---|---|---|
| LLM | GPT / Claude / Gemini | DeepSeek / Qwen / GLM / 文心 / 豆包 |
| LLM 推理 | vLLM (CUDA) | vLLM 国产芯片版 / 飞桨 / MindSpore |
| 向量库 | Qdrant / Milvus | Milvus（国产芯片适配版）/ AnalyticDB Vector |
| 关系库 | Postgres / MySQL | 达梦 / 人大金仓 / OceanBase / GaussDB |
| 缓存 | Redis | 阿里 Tair |
| 消息 | Kafka | Pulsar / RocketMQ |
| K8s | upstream | 华为 CCE / 浪潮云 K8s |
| 观测 | Prometheus / ELK | 国产 APM (听云 / 博睿 / OneAPM) |
| 编程语言 | Python / Go / Java | Python / Go (开源 OK) / Java (Kona JDK) |

## 3. 国密算法

| 算法 | 用途 | 替代 |
|---|---|---|
| **SM2** | 非对称（数字签名） | 替 RSA / ECDSA |
| **SM3** | 哈希 | 替 SHA-256 |
| **SM4** | 对称加密 | 替 AES |
| **SM9** | 标识密码 | 新增 |
| **ZUC (祖冲之)** | 流密码 | - |

实现：
- 软件：国密 OpenSSL
- 硬件：国密 HSM / 加密机（江南天安 / 三未信安）

## 4. AI 推理国产芯片

| 芯片 | 厂商 | 适配状态 |
|---|---|---|
| **昇腾 910B/B2** | 华为 | LLM 推理较成熟 (MindSpore + vLLM-Ascend) |
| **海光 DCU** | 海光 | CUDA 兼容（DTK），主流框架可跑 |
| **寒武纪 MLU** | 寒武纪 | 适配 PyTorch |
| **燧原 GCU** | 燧原 | 适配中 |
| **天数智芯 BI** | 天数智芯 | - |
| **摩尔线程 MTT** | 摩尔线程 | 推理 OK |
| **沐曦 GPU** | 沐曦 | - |

**实战经验**：昇腾 910B 是政企首选；CUDA 兼容性 DCU 较好但供应受限。

- 不做信创首发
- 但**架构保持可适配**：

  - LLM Gateway 抽象（DeepSeek/Qwen 已有）
