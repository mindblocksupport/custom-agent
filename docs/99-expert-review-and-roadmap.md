## 3. 前瞻：12-24 月将发生什么

### 3.1 技术趋势（已可见）
- **A2A 协议成熟**：跨 vendor Agent 协作标准（Google 推动，Apr 2026 v1.0 + Signed Agent Cards）
- **Reasoning model 成本骤降**：Claude / GPT / DeepSeek 思考时间长但 $/思考 token 降
- **百万级上下文常态**：1M 是基础，10M 出现（Gemini 3 Pro）
- **Computer use 跨人类基线**：OSWorld GPT-5.4 75% > 人 72.4%
- **手机 / 车机 端 Agent 成主流**：AutoGLM / Doubao 嵌入车机
- **Voice = 第一公民**：$45B 市场重塑
- **Skill 标准化**：Anthropic Skills 成事实标准
- **MCP / A2A 双协议栈**：MCP 工具 + A2A 协作

### 3.2 商业趋势
- **Outcome-based pricing 主流化**（取代 per-seat）
- **垂直 Agent SaaS 估值碾压横向平台**（Sierra/Cursor/Clay 模式）
- **M&A 浪**：Zendesk → Forethought, Cognition → Windsurf, Google 反向 acquihire
- **企业 AI 治理委员会成 board-level 议题**
- **AI 责任保险市场**（GenAI liability insurance）

### 3.3 监管趋势
- **EU AI Act 高风险 Agent** 合规义务（已生效）
- **网信办大模型备案**+ 算法备案 在国内强制
- **个保法 + 数据安全法**对 Agent 训练 / 推理日益严
- **行业垂直监管**：医疗 / 金融 / 法律各自要求
- **AI Verify 类自评 / 第三方审计成 RFP 必须项**

### 3.4 我们必须为什么做准备
1. **A2A 协议接入 + Signed Agent Card** （L34 待写）
2. **Voice channel 完整支持**（L27 待写）
3. **国密 + 信创全栈适配**（L36 待写）
4. **EU AI Act 合规自评工具**（L8 + L25 加深）

## 7. 当前**最致命的盲点**（必补）

如果只能补 3 个，我建议：

### 1. 🔴 L24 多租户隔离 + L25 数据合规（合并）
**为什么**：所有 To B SaaS 第 1 天就死在这。Glean / Sierra / Decagon 全都重金做这块。**RFP 必填项**。我们目前在 L8 只写了"必须做"——没有可执行设计。

### 2. 🔴 L29 客户 90 天 Onboarding 套件
**为什么**：决定续约率。Sierra 4-6 周，Cresta 4-6 周——业界 benchmark。我们没有 SOP / 工具 / 模板。

### 3. 🔴 L1-L9 基于研究系统重写
**为什么**：现有 9 层是"知识盘点"框架，新研究的真实细节（信创 / DR 数字 / 多 region / connector 详细 / sandbox 选型 / SSO 集成）必须落到这 9 层。否则交付时返工。

## 8. 给老板 / 客户的 Pitch 一页

我们建一个企业 Agent 平台，**4 个真正杀手锏**：
1. **知识工程模块**——把你的 50 个老员工 20 年经验变成可复用 AI（L15）
2. **电脑 Use 能力**——能驱动你 Citrix 老 ERP / 政务系统（L14，下一代 RPA）
3. **完整生命周期管理**——从 Day 1 到 5 年都能可控演进（L23 ADLC + L21 FinOps + L22 HITL）
4. **真正可私有化**——栈 100% OSS / 中性许可，可在你机房 7×24 跑（L16）

**3 大差异化护城河（vs 对手）**：
- Sierra 强在客服垂直，我们强在**知识工程通用**
- Glean 强在搜索，我们强在**Agent 端到端 + 电脑 use**  
- Coze/Dify 是平台 vibe-coding，我们是**企业级 SDLC + 治理**
